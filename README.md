# SDSANet: Slight Deformable Spatial Attention Network for Semantic Segmentation

The implementation of SDSANet in Pytorch. 

## Requirement
```bash
mmsegmentation==0.26.0
mmcv-full==1.6.0
timm == 0.4.12
```
